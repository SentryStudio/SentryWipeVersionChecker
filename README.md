# SentryWipeVersionChecker

🛡️ **SentryWipeVersionChecker** is a lightweight, reliable utility designed specifically to monitor, verify, and check the current version status of the **Sentry Wipe** plugin. 

---

## ✨ Features

* **Real-Time Verification:** Instantly checks the active plugin version against the latest releases.
* **Lightweight & Fast:** Minimal overhead, designed to run seamlessly alongside your existing setup.
